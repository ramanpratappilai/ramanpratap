🌐 Raman Portfolio Website

This is my **personal animated portfolio website** built with **HTML, CSS, and JavaScript**.
It showcases my skills, projects, and contact information in a simple, modern, and responsive design.

* 🎨 Clean and responsive UI design
* ⚡ Smooth scroll animations and typing effect
* 👤 Profile picture upload with local storage persistence
* 📂 Separate files for HTML, CSS, and JavaScript
* 📱 Fully mobile-friendly layout
* 📧 Contact form integration (via Formspree)
